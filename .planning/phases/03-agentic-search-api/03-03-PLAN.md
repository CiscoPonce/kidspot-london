   - Returns venue sponsor details
   - Returns 404 if venue not found
7. Implements /api/sponsors/venues/:id/tier endpoint (admin only):

   - Validates priority (non-negative number)
   - Calls update_sponsor_tier function

8. Implements /api/sponsors/venues/bulk/tier endpoint (admin only):
   - Accepts array of venue updates
   - Validates each venue's tier and priority
   - Calls update_sponsor_tier for each venue
   - Returns summary (updated, errors, total)
9. Implements /api/sponsors/pricing endpoint:
   - Returns static pricing information
   - Includes tier details, features, pricing
