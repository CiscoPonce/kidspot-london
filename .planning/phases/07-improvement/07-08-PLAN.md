<task type="auto">
  <name>Task 1: Create HMAC middleware</name>
  <files>backend/src/middleware/hmac.ts</files>
  <action>
    Create a middleware that verifies HMAC-SHA256 signatures for requests using the shared secret `process.env.INGEST_SIGNING_SECRET`.
    It must expect `x-ingest-signature` and `x-ingest-timestamp` headers.
    Reject requests older than 5 minutes (300 seconds) to prevent replay attacks.
    Calculate the expected HMAC signature using `crypto.createHmac('sha256', secret)` and verify it matches the provided signature using `crypto.timingSafeEqual()`. If the POST request has a body, ensure it is included in the HMAC calculation (e.g., using a raw body buffer). If it's an empty body, handle that gracefully.
  </action>
  <verify>
    <automated>test -f backend/src/middleware/hmac.ts && echo "HMAC middleware created"</automated>
  </verify>
  <done>Middleware correctly rejects missing/invalid signatures or expired timestamps.</done>
</task>
